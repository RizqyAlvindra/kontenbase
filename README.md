## Features
Kontenbase let you focus on the Frontend & Product.
- [x] Hosted Backend
- [x] Auto Generated API & SDK
- [x] Hosted Database 
- [x] Database Migration 
- [x] Authentication and Authorization
- [x] Realtime subscriptions
- [x] Storage
- [ ] Functions (coming soon)
- [ ] Internal Tools (coming soon)
